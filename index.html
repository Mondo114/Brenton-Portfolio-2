Work
Frij'it
Frij’it is a web app to start your day. It’s a collection of customizable widgets (“magnets”) that appear on your fridge (“frij”). You can choose which news, sports, and special interest articles to display. Also available are widgets for stocks, weather, and the time. Ideally there would be dozens of widgets with plenty of customization options.
I created the UI, set up all news-related widgets, and contributed various other parts. If the news articles don’t populate from the dropdown, it’s because the API has hit its limit of uses for the day. Frij’it was a 4-person group effort implemented in about 1 week and utilizes several APIs, Node.js, React, MongoDB, and several npm modules.

Board Game Discovery
Board Game Discovery is a web app that helps you find the game you’re looking to play. If you have a group of people coming over for a game night and you want to know what new game to play, this app allows you to search by different attributes to help narrow it down. You may search by 1 attribute at a time. The reason there are a few dozen games and not thousands in the database is because each game has been entered by hand, allowing for the exact attributes we wanted to include (we couldn’t find already existing APIs that had all the attributes we wanted).
I built the Front End on this project. Board Game Discovery is 3-person group effort web app mostly implemented in 1 week and utilizes Node.js, jQuery, MySQL, and several npm modules.

Firefly Memory Game
Firefly Memory Game is a web game I created. The object is to click on each character once. Each time you click on a character, all twelve characters randomly scramble their position. When you have successfully clicked on each character exactly once, you win. Firefly Memory Game was created over a period of a couple weeks and utilizes Node.js, React, and several npm modules.
